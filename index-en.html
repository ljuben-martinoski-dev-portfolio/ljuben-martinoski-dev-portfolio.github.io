Key Features

  • Driver portal for check-ins (multi-language, reference numbers)
  • Live operator dashboard with drag-and-drop gate assignment
  • Automated gate assignment and status tracking
  • Strict validation to cut reference entry errors

Results

  • Cut check-in time from 2–3 minutes to <1 minute
  • GDPR-compliant with minimal data retention

Enterprise-Grade Password Manager

A secure, local-first password management system implementing industry-standard cryptographic protocols. Prioritizes security, usability, and complete user privacy.

Technical Implementation

  • Encryption: AES-256-CBC via Fernet (authenticated encryption)
  • Key derivation: PBKDF2-HMAC-SHA256 with 480,000 iterations
  • Security: Unique salt per installation, cryptographically secure random generation (secrets module)
  • Architecture: Modular design with separation of concerns (encryption, storage, UI)
  • Deployment: Standalone executable via PyInstaller for cross-machine portability

Features

  • Password generation, strength analysis algorithm, category management
  • Encrypted backup/restore, comprehensive error handling, professional terminal UI
Python 3.14 Cryptography (Fernet) PyInstaller Colorama Tabulate
