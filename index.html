LagerverwaltungsApp (v1.5.0)

Echtzeit-Torverwaltung für Lager mit Fahrer-Self-Service-Portal und Operator-Dashboard.

Hauptfunktionen

  • Fahrerportal für Check-ins (mehrsprachig, Referenznummern)
  • Live-Dashboard mit Drag-and-Drop-Torzuweisung
  • Automatische Torvergabe und Status-Tracking
  • Strenge Validierung reduziert Eingabefehler

Technischer Stack

  • Backend: Python FastAPI, SQLAlchemy, WebSockets, SQLite
  • Frontend: React 18 + TypeScript + Vite, Tailwind CSS, Zustand
  • Desktop: Electron-Wrapper für portable Windows Builds

Ergebnisse

  • Check-in-Zeit von 2–3 Minuten auf <1 Minute reduziert
  • DSGVO-konform mit minimaler Datenspeicherung
  • Offline-fähige lokale Bereitstellung, kein Cloud-Zwang
  • Einsatzbereit und braucht noch testing ins Feld
React 18 TypeScript Python FastAPI SQLAlchemy WebSockets Electron Tailwind CSS SQLite

Enterprise-Grade Password Manager

Sicheres, lokal-first Passwortverwaltungssystem mit branchenüblichen Krypto-Verfahren. Fokus auf Sicherheit, Benutzerfreundlichkeit und vollständige Privatsphäre.

Technische Umsetzung

  • Verschlüsselung: AES-256-CBC via Fernet (authentifizierte Verschlüsselung)
  • Schlüsselableitung: PBKDF2-HMAC-SHA256 mit 480.000 Iterationen
  • Sicherheit: Eindeutiges Salt pro Installation, kryptografisch sichere Zufallserzeugung (secrets-Modul)
  • Architektur: Modulares Design mit Trennung (Verschlüsselung, Speicher, UI)
  • Deployment: Standalone-Executable via PyInstaller für portablen Einsatz

Funktionen

  • Passwort-Generierung, Stärke-Analyse, Kategorieverwaltung
  • Verschlüsselte Backup- und Wiederherstellungsfunktionen
  • Umfassende Fehlerbehandlung, professionelle Terminal-UI
Python 3.14 Cryptography (Fernet) PyInstaller Colorama Tabulate
